# Vendored fonts

## OpenDyslexic (opendyslexic-400.woff2, opendyslexic-700.woff2)

- **What:** OpenDyslexic — a typeface designed to increase readability for
  readers with dyslexia. Offered as an optional "Dyslexia-friendly font"
  reading aid in the Office/DOCX reader.
- **License:** SIL Open Font License 1.1 (OFL-1.1) — free to bundle and
  redistribute; OFL is compatible with this project's AGPL-3.0.
- **Source:** https://opendyslexic.org  ·  packaged via
  https://www.npmjs.com/package/@fontsource/opendyslexic (v5.2.5, latin subset).
- **Used by:** `static/css/readers.css` (`@font-face`) + the View ▾ menu's
  "Dyslexia-friendly font" toggle in `static/js/readers/office.js`.

Weights: 400 (regular) and 700 (bold). Vendored verbatim.
